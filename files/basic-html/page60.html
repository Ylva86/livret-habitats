Comment construire les abris ? Combien d%u2019abris %u00e0 chauve-souris ? - les chauve-souris %u00e9tant gr%u00e9gaires, l%u2019id%u00e9al est d%u2019installer un bon nombre d%u2019abris pour augmenter les chances de voir nos amies s%u2019installer ; - il est recommand%u00e9 d%u2019installer plusieurs nichoirs dans des conditions vari%u00e9es afin de convenir au maximum de chauve-souris ; - attention : les abris ne remplacent pas les g%u00eetes naturels mais permettent d%u2019aider les chauve-souris face %u00e0 la r%u00e9duction de leurs habitats.Quels mat%u00e9riaux ? Du bois similaire %u00e0 celui des nichoirs %u00e0 oiseaux : - planches de bois massif (%u00e9paisseur de 1,5 %u00e0 2 cm) r%u00e9sistant %u00e0 l%u2019humidit%u00e9, non trait%u00e9 : pin, sapin, m%u00e9l%u00e8ze, c%u00e8dre rouge, peuplier, ch%u00eane, aulne%u2026 ; - laisser l%u2019int%u00e9rieur brut : ne pas poncer le bois ; - badigeonner l%u2019ext%u00e9rieur %u00e0 l%u2019huile de lin ou cire d%u2019abeille pour %u00e9tanch%u00e9ifier, ne pas peindre ni r%u00e9aliser de traitement chimique. On peut appliquer du brou de noix pour foncer la couleur.
